# QR Payment Feature - План проекта

## Обзор

**Название:** QR-код оплата ваучеров  
**Версия:** 1.0  
**Дата начала:** 24.12.2024  
**Статус:** В разработке

## Описание

Добавление возможности продажи ваучеров через QR-коды на физических карточках. Клиент сканирует QR-код, оплачивает товар, получает ваучер в кошелёк.

> [!IMPORTANT]
> **Фаза 1** — это полная настройка и создание всей механики нового способа продаж (QR-ссылки, попытки оплаты, логика создания чеков и продаж).
> **Фаза 2** (подключение реальных систем оплаты Click/Payme) будет выполняться согласно первоначальным инструкциям интеграции, дорабатывая созданную в Фазе 1 механику.

## Риски

| Риск | Вероятность | Влияние | Митигация |
|------|-------------|---------|-----------|
| Нехватка ваучеров при оплате | Средняя | Высокое | Проверка перед оплатой |
| Таймаут платёжной системы | Низкая | Высокое | 2-минутный таймаут + retry |
| Дублирование транзакций | Низкая | Критическое | Уникальные ID + проверка статуса |
