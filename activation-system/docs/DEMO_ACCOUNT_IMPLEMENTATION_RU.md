## 🔧 Как это работает

### 1. При запросе SMS-кода (`sendOtp`)
- Система проверяет, является ли номер демо-аккаунтом (+998003332211)
- Если ДА:
  - Использует фиксированный код `777777` вместо случайного
  - **НЕ отправляет SMS** через Eskiz (экономия SMS и работает без интернета)
  - Помечает запись в базе как `demo-account-appstore`
  - Логирует событие в консоль
- Если НЕТ:
  - Работает как обычно (генерирует случайный код и отправляет SMS)

### 2. При проверке кода (`verifyOtp`)
- Система проверяет, является ли номер демо-аккаунтом
- Если ДА:
  - **Код не истекает** (обычно коды действуют 5 минут)
  - Можно использовать один и тот же код многократно
  - При создании клиента логирует в консоль
- Если НЕТ:
  - Работает как обычно (код действителен 5 минут)

## 📱 Что можно делать с демо-аккаунтом

### ✅ МОЖНО:
1. **Авторизоваться** в любом магазине (store)
2. **Активировать ваучеры** - введите код ваучера
3. **Просматривать историю** активаций
4. **Добавлять в Wallet** - Apple Wallet / Google Wallet
5. **Получать SMS** - если захотите (можно включить отправку для демо-номера)
6. **Делать продажи** на этот номер:
   - Мерчант может продать ваучер на +998003332211
   - Клиент получит ваучер и сможет его активировать

### ✅ Демо-клиент в базе данных
При первой авторизации создается настоящая запись в таблице `Client`:
```
phoneNumber: +998003332211
id: автоматический
createdAt: дата создания
```

Этот клиент работает как обычный:
- К нему привязываются ваучеры через `OnlineVoucher`
- Записываются активации в `VoucherActivation`
- Сохраняется история

## 🗑️ Как удалить после App Store Review

После одобрения приложения можно:

### Вариант А: Удалить код полностью
Удалите строки 84-87 и 102-129 из `storeController.js`

### Вариант Б: Отключить через переменную окружения
Добавьте в `.env`:
```
ENABLE_DEMO_ACCOUNT=false
```

И измените код:
```javascript
const DEMO_ENABLED = process.env.ENABLE_DEMO_ACCOUNT === 'true';
const isDemo = DEMO_ENABLED && phone === DEMO_PHONE;
```

### Вариант В: Оставить для тестирования
Можно оставить демо-аккаунт - он не влияет на работу системы

## 🔒 Безопасность

- Демо-аккаунт работает только для одного конкретного номера
- Не влияет на отправку SMS для других пользователей
- Все действия логируются
- Можно отслеживать по записям `demo-account-appstore` в базе данных
